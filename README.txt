# Menambahkan Nasabah
	1. Login admin
	2. Pilih 1 pada menu Admin
	3. Masukkan nama pertama nasabah
	4. Masukkan nama kedua nasabah
	5. Masukkan password yang akan digunakan nasabah untuk login nanti
	6. Masukkan Nomor rekening yang maksimal 8 digit
	7. Nasabah berhasil ditambahkan!
#
# Menghapus Nasabah
	1. Login admin
	2. Pilih 2 pada menu Admin
	4. Masukkan data nasabah yang ingin dihapus dengan benar
	5. Nasabah berhasil di hapus!
#
# Menyetorkan uang nasabah kedalam bank
	1. Login admin
	2. Pilih menu 3 pada menu Admin
	3. Masukkan data nasabah yang setoran
	4. Masukkan nominal setoran
	5. Saldo nasabah berhasil diperbarui!
#
# Sorting Nasabah
	1. Login admin
	2. Pilih menu 4 pada menu Admin
	3. Pilih urutan sorting yang diinginkan!
#
# Mencari Nasabah
	1. Login Admin
	2. Pilih Menu 5 pada menu Admin
	3. Pilih kategori pencarian
	4. Hasil pencarian akan ditampilkan pada console
#
#
#
#	||Pemakaian Untuk Nasabah||
#
# Withdraw Uang
	1. Login nasabah
	2. Pilih menu 1 pada menu Nasabah
	3. Masukkan nominal withdraw
	4. Withdraw berhasil!
#
# Transfer Uang
	1. Login nasabah
	2. Pilih menu 2 pada menu Nasabah
	3. Masukkan rekening penerima
	4. Masukkan nominal transfer
	5. Transfer Berhasil!
